===============
User ohne Shell
===============

:Authors: - Uwe Müller
:Date:   2013-31-08

Allgemein
=========

- wird durch den Paket-Admin angelegt,
- hat keine Shell (das Programm ``passwd`` wird an Stelle einer Shell gestartet).

Rolle
-----

- als E-Mail-Account.

Rechte
------

- nur Passwortänderung möglich.
